MINHAS ESPECIALIDADES

Programação Orientada a Objetos

Com o avanço da tecnologia, é importantíssimo nos mantermos atualizado quanto a forma que programamos nossas aplicações. O conceito de Programação Orienta a Objetos já existe há alguns anos, e é ótima quando falamos de otimização, segurança, a eliminação de redundâncias, e flexibilidade. Tive meu primeiro contato com ela na faculdade (2022) e a venho empregando até hoje.

PHP

Essa é linguagem com que eu mais tenho contato, e é a linguagem que eu mais domino atualmente na programação de aplicações, utilizo ela desde 2019! Podendo ser aplicada de forma orientada a objetos ou não, é uma linguagem muito flexível e com um vasto conjunto de ferramentas, para manipular arquivos, tratamento de dados e trabalhar de forma segura com dados sigilosos.

MySql

Hoje em dia, é essencial saber armazenar e utilizar os dados que são gerados nas aplicações, para isso o MySql é ótimo, junto com o PHP, se tornam ferramentas excepcionais. Como o PHP, utilizo essa ferramenta desde 2019! Ela serve para armazenar os dados dos usuários, dados que são gerados durante utilização de aplicações e dados sigilosos. E com ela é possível armazenar diversos tipos de dados, números inteiros, flutuantes, strings e etc.
